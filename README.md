# LeetMentor AI – Personalized DSA Progress Analyzer

LeetMentor AI is a personalized DSA (Data Structures & Algorithms) progress tracking system that helps users analyze their problem-solving performance and improve coding skills through structured insights.

---

## 📌 Features

- Tracks DSA problem-solving progress
- Analyzes patterns in solved problems
- Identifies weak and strong topic areas
- Provides personalized improvement insights
- Helps build consistent problem-solving habits

---

## 🧠 Motivation

Most students solve DSA problems without structured feedback.  
LeetMentor AI aims to provide **data-driven insights** to help users improve efficiently instead of random practice.

---

## ⚙️ Tech Stack

- Programming Language: C++ / JavaScript (as applicable)
- Backend: Node.js (if used)
- Database: MongoDB (if used)
- Concepts: Data Structures, Algorithms, Data Analysis

---

## 📊 How It Works

1. User submits or logs solved problems
2. System categorizes problems by topic
3. Performance is analyzed over time
4. Weak areas are highlighted
5. Suggestions are generated for improvement
